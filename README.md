# Cach Mobile App

Cach is a mobile application for tokenizing digital assets and participating in crowdfunding campaigns.

## Features

- User authentication
- Asset tokenization
- Marketplace for trading tokenized assets
- Wallet management
- Crowdfunding campaigns

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Expo CLI
